Unit activities

So for this activity, and this is the encryption algorithm I will probably put in place in my system implementation, I used a monoalphabetic cipher. This one will generate a key which will be different every time the algorithm is used and an encrypted message. The communication between my server and the main application will be symmetrically encrypted. I found this method effective and sufficient for my application, as asymmetric is more computationally demanding and the ISS resources are rather limited.

This is the proposed algorithm by Tutorials Point India Private Limited (n.d.):

								
from string import ascii_letters, digits
from random import shuffle


def random_monoalpha_cipher(pool=None):
    if pool is None:
        pool = ascii_letters + digits
    original_pool = list(pool)
    shuffled_pool = list(pool)
    shuffle(shuffled_pool)
    return dict(zip(original_pool, shuffled_pool))


def inverse_monoalpha_cipher(monoalpha_cipher):
    inverse_monoalpha = {}
    for key, value in monoalpha_cipher.items():
        inverse_monoalpha[value] = key
    return inverse_monoalpha


def encrypt_with_monoalpha(message, monoalpha_cipher):
    encrypted_message = []
    for letter in message:
        encrypted_message.append(monoalpha_cipher.get(letter, letter))
    return ''.join(encrypted_message)


def decrypt_with_monoalpha(encrypted_message, monoalpha_cipher):
    return encrypt_with_monoalpha(
        encrypted_message,
        inverse_monoalpha_cipher(monoalpha_cipher)
    )


if __name__ == "__main__":
    key = random_monoalpha_cipher()
    message = input("What do you want to encrypt?")
    encrypted_message = encrypt_with_monoalpha(message, key)
    print(encrypted_message)
    print(decrypt_with_monoalpha(encrypted_message, key))

The key will essentially be a dictionary with their paired equivalent and the message encoded according to that key. This same key will be reversed and used in the decoding.

Alongside hashing, cryptography will help the application adhere to GDPR standards by protecting the user’s information from malicious users and preventing tampering. The application will also let users delete their account data any time they want to.
